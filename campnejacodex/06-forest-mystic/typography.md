# Typography System

- **Display:** Lora — Mystic hero type
- **Secondary:** Space Grotesk — Labels
- **Body:** Manrope — Body copy

Usage hierarchy:
1. Display type for hero headlines and large callouts.
2. Secondary type for section headers, numerals, and nav labels.
3. Body type for paragraphs, captions, and list content.

Responsive behavior: scale display type from 48px desktop hero down to 28px on mobile; body maintains 18px desktop, 16px mobile with 150% line height.
