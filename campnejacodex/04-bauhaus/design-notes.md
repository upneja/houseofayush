# Bauhaus Modernist Direction

## Hero + Layout
- Hero image: Volleyball court beside the pavilion awaiting day tournaments. (campnejapics/volleyball.jpg) with thematic treatments.
- Layout grid: 12-column rigid grid with modular 40px baseline and strict alignments.
- Section styling: Stacked geometric bands separated by angled dividers.

## Color & Texture
- Palette documented in palette.json
- Textures/background: Flat colors with halftone noise blocks.

## Typography
- Display: Futura Condensed — Bold headlines
- Secondary: Montserrat — Labels
- Body: IBM Plex Sans — Copy

## Iconography + Logo
- Iconography direction: Abstract pictograms built from circles and rectangles.
- Logo/wordmark exploration: Blocky CN monogram inside a circle-square lockup.

## Component Library
Pill buttons with offset drop shadows, cards as grid modules, nav as sticky bar, dividers as diagonal bars, gallery as overlapping rectangles, CTA as oversized Bauhaus poster.

## Animation / Motion
- Slide-in panels, snapping transitions mimicking mechanical motion.
