# Typography System: Luxury Travel

## Typefaces

### Display
**Gilda Display** - Elegant, refined serif
- H1, H2

### Body
**Karla** - Clean, modern sans
- Body text

## Type Scale
```
H1: 72px / Gilda Display / 400
H2: 48px / Gilda Display / 400
Body: 17px / Karla / 400
```
