# Typography System: High Fashion Editorial

## Typeface Strategy
Ultra-minimal, editorial precision with extreme contrast and generous whitespace.

## Primary Typefaces

### Display / Headlines
**Bodoni Moda** (or Didot)
- Weights: 400, 500, 600, 700
- Usage: H1, H2, hero titles
- Characteristics: High-contrast serifs, editorial luxury

### Subheadings
**Playfair Display**
- Weights: 400, 500, 700
- Usage: H3, H4, section subtitles
- Characteristics: Elegant transitional serif

### Body / Content
**Inter** (or Helvetica Neue)
- Weights: 300, 400, 500
- Usage: Body copy, descriptions
- Characteristics: Clean sans-serif, maximum readability

### Labels / UI
**Futura PT** (or Montserrat)
- Weights: 300, 400, 500, 700
- Usage: Navigation, buttons, captions
- Characteristics: Geometric sans, refined

## Type Scale

```
H1 (Hero):           120px / 7.5rem    | Bodoni  | 400 | -0.02em
H2 (Section):        72px / 4.5rem     | Bodoni  | 500 | -0.01em
H3 (Subsection):     48px / 3rem       | Playfair | 500 | 0em
H4 (Card):           32px / 2rem       | Playfair | 600 | 0em
Body Large:          20px / 1.25rem    | Inter   | 300 | 0em
Body:                16px / 1rem       | Inter   | 400 | 0.01em
Body Small:          14px / 0.875rem   | Inter   | 400 | 0em
Label:               12px / 0.75rem    | Futura  | 500 | 0.15em (uppercase)
Caption:             11px / 0.6875rem  | Futura  | 400 | 0.05em
```

## Line Heights
- Display: 1.0 (ultra-tight)
- Headings: 1.2
- Body: 1.8 (generous)
- Labels: 1.5

## Special Treatments

### Oversized Typography
- Key phrases: 10-15vw font size
- Creates editorial impact
- Overlaps images slightly

### Vertical Type
- Sidebar dates/labels
- Rotated -90deg
- Futura uppercase, 11px

### Kerning Perfection
- Manual letter-spacing adjustments
- Bodoni requires careful kerning
- Numbers: Tabular spacing

### Minimal Punctuation
- Em dashes without spaces
- Periods omitted on headings
- Quotation marks: Minimal use

## Grid & Alignment
- Everything aligned to strict grid
- Left-align for body
- Center-align for displays (sparingly)
- Right-align for accents

## Responsive
- Mobile: Reduce display by 50-60%
- Maintain proportional relationships
- Never compromise whitespace
