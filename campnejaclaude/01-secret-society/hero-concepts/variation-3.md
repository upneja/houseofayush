## Details

### Background
- NO photo in hero
- Solid obsidian (#0A0A0F)
- Subtle paper grain texture (5% opacity)
- Faint radial gradient (center lighter by 3%)
- Creates intimate, exclusive feeling

### Decorative Lines
- Top and bottom horizontal dividers
- Width: 80% of container
- Height: 2px double border style
- Color: Ceremonial gold
- Ornamental caps: Small circles at ends (8px diameter)
- Creates frame for content

### Typography
- Event name: Cormorant Garamond 6rem / 300 (thin weight!)
- Color: Ceremonial gold
- Letter spacing: 0.15em (very wide)
- Text transform: uppercase
- Dates: JetBrains Mono 1.25rem / 500
- Color: Pale gold
- Letter spacing: 0.2em
- Positioned directly under name (tight spacing)

### Tagline
- Crimson Text 1.25rem / 400 italic
- Color: Parchment with 80% opacity
- Max width: 600px
- Centered
- Positioned between dividers

### CTA
- Single primary button
- Minimal size: 14px × 36px padding
- Clean, refined
- Centered below tagline
- 2rem margin top

### Optional Elements
- Small geometric sigil watermark (15% opacity) behind text
- Very large, faint, centered
- Does not distract from typography

### Animation Sequence
1. Texture fade in (0.4s)
2. Top divider: Draw from center outward (1s ease)
3. Event name: Letter-by-letter fade (1.2s, 0.5s delay)
4. Dates: Fade in (0.4s, 1s delay)
5. Bottom divider: Draw from center outward (1s, 1.2s delay)
6. Tagline: Fade in (0.6s, 1.5s delay)
7. Button: Fade + slight lift (0.5s, 1.8s delay)

### Interaction
- Hover on hero area: Subtle sigil watermark rotates slowly
- Scroll down: Entire hero fades out elegantly
- Very refined, no heavy effects

### Height
- Desktop: 85vh (not full viewport)
- Creates anticipation for content below
- Tablet: 75vh
- Mobile: 70vh

### Color Variations
- Could use deep purple gradient instead of pure obsidian
- Or subtle crimson undertone (5% blend)
- Maintains darkness while adding mystique

### Responsive Behavior
- Mobile: Event name reduces to 3.5rem
- Letter spacing reduces to 0.1em
- Dividers: 90% width
- All else maintains proportions
- No layout changes, just scaling

### Use Case
- Perfect for sophisticated, exclusive feel
- Emphasizes typography and negative space
- Less about venue, more about mystery
- Complements photo-heavy sections below
- "Less is more" ceremonial aesthetic
