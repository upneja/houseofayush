# Typography System: Mystical Forest Cult

## Typefaces

### Display
**Uncial Antiqua** - Ancient, mystical, rune-like
- H1, Event name

### Headers
**Cinzel** - Imperial, mysterious
- H2, H3, section titles

### Body
**Lato** - Clean, readable contrast
- Body text

## Type Scale
```
H1: 96px / Uncial Antiqua / uppercase
H2: 56px / Cinzel / 600
Body: 18px / Lato / 300
```

## Style
- Glowing text effects (bioluminescent green)
- Mystical drop shadows
- Rune symbols as decorative elements
- Fog/mist overlays on text
