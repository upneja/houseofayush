# Typography System: Fantasy Noble House

## Typeface Strategy
Medieval authority meets modern readability with strong serifs and heraldic character.

## Primary Typefaces

### Display / Headers
**Cinzel Decorative** (or Trajan Pro)
- Weights: 400, 700, 900
- Usage: H1, H2, event name, section titles
- Characteristics: Classical Roman capitals with imperial presence

### Subheadings
**Libre Baskerville** (or Lora)
- Weights: 400, 400i, 700
- Usage: H3, H4, card titles
- Characteristics: Strong serifs with medieval book quality

### Body / Content
**Source Serif Pro** (or PT Serif)
- Weights: 400, 400i, 600, 700
- Usage: Body copy, descriptions, lists
- Characteristics: Highly readable serif with historical feel

### Labels / UI
**Montserrat** (uppercase)
- Weights: 500, 600, 700
- Usage: Buttons, labels, navigation
- Characteristics: Modern sans-serif for clean UI contrast

### Accent / Decorative
**UnifrakturMaguntia** (sparingly!)
- Weight: 400
- Usage: Drop caps, decorative initials only
- Characteristics: Gothic blackletter for medieval flavor

## Type Scale

```
H1 (Hero Title):     84px / 5.25rem   | Cinzel Dec | 900 | 0em (uppercase)
H2 (Section):        56px / 3.5rem    | Cinzel Dec | 700 | 0.02em
H3 (Subsection):     40px / 2.5rem    | Libre Bask | 700 | 0em
H4 (Card Title):     28px / 1.75rem   | Libre Bask | 700 | 0em
Body Large:          20px / 1.25rem   | Source     | 400 | 0em
Body:                18px / 1.125rem  | Source     | 400 | 0em
Body Small:          16px / 1rem      | Source     | 400 | 0em
Label/CTA:           14px / 0.875rem  | Montserrat | 600 | 0.1em (uppercase)
Caption:             14px / 0.875rem  | Source     | 400 | 0em
```

## Line Heights
- Display (Cinzel): 1.1
- Headings (Libre): 1.3
- Body (Source): 1.7
- Labels (Montserrat): 1.4

## Special Treatments

### House Crests
- Accompanied by small caps labels
- Montserrat 600, 12px, uppercase, 0.15em

### Banner Headers
- Cinzel Decorative with decorative underlines
- Gold accent line (3px) below
- Small diamond ornaments (◆) flanking

### Drop Caps
- First paragraph: UnifrakturMaguntia 4x body size
- Heraldic gold color (#C9A961)
- Float left with margin

### Numbered Lists
- Roman numerals (I, II, III)
- Cinzel 700, heraldic gold
- Aligned right in left margin

## Heraldic Elements

### Section Dividers
- ══════ ◆ ══════
- Double lines with center diamond
- Heraldic gold color

### Quotations
- Italic Source Serif with left border
- Banner red accent bar (4px)
- Increased left padding

## Responsive Adjustments
- Mobile: Display sizes reduce 40%
- Maintain uppercase Cinzel for authority
- Preserve heraldic decorative elements
- Simplify ornamental treatments
