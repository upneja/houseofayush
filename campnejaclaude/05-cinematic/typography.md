# Typography System: Cinematic Universe

## Typefaces

### Display
**Bebas Neue** - Bold, cinematic, poster-style
- Weights: 400, 700
- H1, H2, dramatic headings

### Serif
**Lora** - Elegant serif for contrast
- Weights: 400, 500, 700
- H3, H4, subtitles

### Body
**Roboto** - Clean, professional
- Weights: 300, 400, 700
- Body text, descriptions

## Type Scale
```
H1: 108px / Bebas Neue / 700 / uppercase
H2: 72px / Bebas Neue / 400 / uppercase
H3: 36px / Lora / 600
Body: 18px / Roboto / 300
```

## Style
- Dramatic scale contrast
- Metallic/gradient text effects
- Cinematic letter-spacing
- All-caps for impact moments
