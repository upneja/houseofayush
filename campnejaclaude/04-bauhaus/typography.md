# Typography System: Bauhaus Modernist

## Typefaces

### Display
**Futura** - Geometric sans-serif, Bauhaus classic
- Weights: 300, 500, 700, 900
- H1, H2, event name

### Body
**DM Sans** - Clean, modern, readable
- Weights: 400, 500, 700
- Body copy, descriptions

## Type Scale
```
H1: 96px / Futura / 900 / uppercase
H2: 60px / Futura / 700
H3: 40px / Futura / 500
Body: 18px / DM Sans / 400
Label: 14px / Futura / 700 / uppercase / 0.1em spacing
```

## Style
- High contrast: Large display vs. small body
- Geometric precision
- Perfect alignment to grid
- Primary colors for emphasis
