# Typography System: Cabin Outdoor Brand

## Typefaces

### Display
**Oswald** - Bold, outdoor brand headers
- Weights: 400, 600, 700
- H1, H2

### Body
**Open Sans** - Clean, readable, modern outdoor
- Weights: 400, 600, 700
- Body, descriptions

## Type Scale
```
H1: 72px / Oswald / 700 / uppercase
H2: 48px / Oswald / 600
Body: 18px / Open Sans / 400
Label: 14px / Oswald / 600 / uppercase
```

## Style
- Outdoor brand aesthetic
- Badge/patch typography
- Topographic line backgrounds
- Trail marker elements
