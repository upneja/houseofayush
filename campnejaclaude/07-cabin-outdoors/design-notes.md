# Cabin Outdoor Brand Design Concept

## Visual Direction
**Theme:** REI/Patagonia earth tones, topographic maps, outdoor badges
**Mood:** Adventurous, rugged, authentic, natural
**Reference:** Outdoor brand catalogs, National Parks, trail guides

## Color Strategy
Forest greens, earth tones (terracotta, sandstone), warm neutrals. Sky blue and sunset orange accents.

## Layout
- Topographic line patterns as backgrounds
- Badge/patch style elements
- Trail marker navigation
- Grid layouts like gear catalogs
- Photo-heavy sections

## Components
- **Buttons:** Patch-style with border, outdoor badge feel
- **Cards:** Like gear cards, earth-toned borders
- **Navigation:** Trail marker style
- **Badges:** Circular outdoor emblems
- **Icons:** Simple line icons (mountains, trees, tents)

## Effects
- Topographic line animations
- Elevation profile graphics
- Trail map aesthetics
- Vintage outdoor poster vibes
