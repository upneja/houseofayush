# Festival Brand Design Concept

## Visual Direction
**Theme:** Sunset gradients, stickers, playful typography
**Mood:** Energetic, joyful, colorful, celebratory
**Reference:** Music festivals, Coachella, summer festival branding

## Color Strategy
Vibrant sunset gradients (pink to orange to yellow). Electric accents. Playful, high-energy colors throughout.

## Layout
- Colorful gradient backgrounds
- Sticker-style elements scattered
- Playful asymmetric layouts
- Photo frames with colorful borders
- Overlapping elements

## Components
- **Buttons:** Gradient fills, rounded, playful
- **Cards:** Colorful borders, sticker aesthetic
- **Navigation:** Floating pill-style
- **Stickers:** Decorative scattered elements
- **Badges:** Festival pass style

## Effects
- Gradient animations
- Playful hover bounces
- Confetti/celebration particles
- Rotating sticker elements
- Vibrant shadows and glows
