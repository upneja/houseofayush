# House of Ayush — Visual Identity System

A luxury fashion house brand identity inspired by Dior, Celine, and Saint Laurent.

## Design Philosophy

**Elegant. Editorial. Confident. High-End.**

This visual identity system embodies timeless luxury through:
- Ultra-minimal aesthetics
- Premium typography
- Monochromatic sophistication
- Artisanal details

---

## Brand Elements

### 🎨 Color Palette

**Monochrome Foundation:**
- Black: `#0a0a0a`
- Charcoal: `#1a1a1a`
- Gray: `#666666`
- White: `#fafafa`

**Accent Color:**
- Warm Taupe/Cognac: `#8b7355` — Used sparingly for emphasis and luxury details

### ✍️ Typography

**Serif (Headlines & Display):**
- Playfair Display / Cormorant Garamond
- Weights: 300 (Light), 400 (Regular)
- Letter-spacing: 0.15em - 0.35em
- Usage: Wordmark, headlines, editorial quotes

**Sans-Serif (Body & UI):**
- Inter
- Weights: 300 (Light), 400 (Regular), 500 (Medium)
- Letter-spacing: 0.05em - 0.25em
- Usage: Navigation, body text, labels

### 🏷️ Wordmark

The House of Ayush wordmark features:
- Stacked layout: "HOUSE OF" above "AYUSH"
- Horizontal divider line
- Accent dot detail
- Ultra-wide letter-spacing
- Refined, editorial feel

**Files:**
- `logo.svg` — Scalable vector wordmark

## Usage

1. Open `index.html` in a modern web browser
2. Scroll through the sections to experience the full visual identity:
   - Hero section with wordmark
   - Studio photography aesthetic
   - Editorial layouts
   - Material showcases
   - Typography specimens
   - Monochrome gallery
   - Brand philosophy

### Customization

**Change the accent color:**
Edit the `--accent` variable in `styles.css`:
```css
--accent: #8b7355; /* Current: Warm taupe */
```

**Adjust typography:**
Modify font families in `:root`:
```css
--font-serif: 'Playfair Display', serif;
--font-sans: 'Inter', sans-serif;
```

**Spacing system:**
Use CSS custom properties for consistent spacing:
```css
--space-xs: 0.5rem;
--space-sm: 1rem;
--space-md: 2rem;
--space-lg: 4rem;
--space-xl: 6rem;
--space-xxl: 10rem;
```
